# Sample Library

Add WAV files here for the sampler:

- `drum-break.wav` - A classic drum break for chopping practice
- `vocal-chop.wav` - Vocal sample for pitch and time manipulation
- `bass-hit.wav` - Bass hit for layering exercises
- `piano-loop.wav` - Piano loop for melodic sampling

## Recommended Sample Specifications

- Format: WAV (16-bit or 24-bit)
- Sample Rate: 44.1kHz or 48kHz
- Duration: 1-8 seconds (longer for loops)
- Mono or Stereo

## Sources for Royalty-Free Samples

- [Freesound.org](https://freesound.org)
- [Splice](https://splice.com) (subscription)
- Record your own!
